
# Course Syllabus - STAT 5014 Fall 2026

## Course Information

**Instructor:** Dr. Michael Schwob (`schwob@vt.edu`)

**Communication:** The best way to reach me is through email (please
include STAT 5014 in the subject line).

**Office Hours:** By appointment (Hutcheson 210-B)

**Meeting Location/Time:** SEITZ 313, Wednesdays 9:05AM - 9:55AM

**Description:** This one-credit-hour course serves as an introduction
to various aspects of statistical programming for first-year statistics
graduate students. It will provide an overview of relevant tools (such
as Git, LaTeX, R, and python) but will not dive into depth.

**Prerequisites:** None. Well, ownership of a computer helps.

**Objectives:** Students will be introduced to many of the tools
required for modern day statistical programming. Students will also gain experience working with these tools
through regular homework assignments.

**Structure:** This is a synchronous in-person class. Regular attendance
is expected. Lectures will not be recorded. Active participation in
class is strongly encouraged; please bring your personal laptop
to class so you can run code as we go.

**Software:** The following software tools will be necessary for this

-   [Git](https://git-scm.com/)

-   [LaTeX](https://miktex.org/) - many machines have a version already
    installed
-   [R](https://www.r-project.org/)
-   [python](https://www.python.org/) - most machines have a version

We will use the following IDEs to code in R and Python:

-   [Rstudio](https://posit.co/download/rstudio-desktop/) for R - for

We will use Markdown for lecture notes, which you will soon learn how to generate!

## Grading

This is a **pass/fail** course. To pass this course, students must pass
at least 5 of the 6 homework assignments. Attendance is expected but is
not a component of the final grade.

## Homework

There will be six homework assignments. Homework will be submitted through
GitHub. Students will create their own repositories for this course
where they will push all homework submissions. Details will be provided
in Homework 0 (ungraded but required).

Students are encouraged to collaborate when appropriate, but each
student must complete and submit their own individual work.

**Late submissions will not be considered unless the instructor grants
specific approval.**

The use of generative AI **is** permitted in this course, but you must be able to explain everything you submit. Use AI to learn, not to replace you.
